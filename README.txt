TERAS UNIVERSAL — Hero + Corporate Capability Statistics Update

1. Ekstrak fail ZIP ke dalam:
   C:\WEBSITE\terasuniversal-website

2. Buka PowerShell dalam folder tersebut.

3. Jalankan:
   powershell -ExecutionPolicy Bypass -File .\apply-hero-corporate-stats-update.ps1

4. Uji:
   npm run build

5. Commit:
   git add .
   git commit -m "Upgrade hero section and corporate capability statistics"
   git push origin main

Nota:
- Angka yang digunakan ialah angka struktur yang telah disahkan: 4 pilar, 7 tahap metodologi dan 3 mod penyampaian.
- Tiada angka peserta, klien atau tahun pengalaman yang belum disahkan dimasukkan.
